# vue单词

## 第三单元

**内置指令总结**

| 指令 | 含义 | 快速记忆 |
| :--- | :--- | :--- |
| v-bind | 进行属性值的绑定 |  |
| v-model | 进行表单数据的双向绑定 |  |
| v-else   v-show   v-else-if v-else | 进行元素的显示和隐藏转化 | 原生js的条件判断 if else if  else |
| v-for | 列表/表格渲染 | 原生js的for循环 |
| v-on | 进行事件的绑定 |  |
| v-text | 进行文本内容的绑定 | text指的是文本 |
| v-html | 进行页面元素的绑定 | html标签 |
| v-once | 只绑定一次 |  |
| v-pre | 跳过\{\{\}\}的解析,写什么,就显示什么 |  |
| v-cloak | 需要配合样式使用,作为简单了解 |  |
| 自定义指令 |  |  |
| v-自定义指令名 |  |  |
